# Reunite

Reuniting families separated by natural disaster, using an intelligent filtering system powered by Watson Visual Recognition

## Development

### Running Code

#### App

```bash

npm start

```

#### Server

```bash

cd server
npm start

```

#### Client

```bash
cd client
npm start
```

### Running Tests

#### Server

```bash
cd server
npm test
```

#### Client

```bash
# Cypress Tests
Make sure the app is running
npm run cypress:open

# Unit Tests
cd client
jest
```
